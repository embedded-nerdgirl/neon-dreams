Quick little readme! This directory contains little Lua tools mostly for creating
and formatting input data to a JSON for placement in ./assets/json/*** or other
relevant places. manifest-tool is not going in here.

There is also an XOR tool for encrypting future *shader files*. The key is not secure,
in fact, the key is "0xC0C1EFAA0D16FC77" -- if you want to un-XOR encrypted data, there
you go, enjoy. This key is mostly for Fragment Shaders located in ./assets/shaders
